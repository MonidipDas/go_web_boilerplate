# Go Web Boilerplate

## Run the server

From the project root:

```powershell
cd "c:\Users\MONIDIP DAS\OneDrive\Desktop\go-web boilerplate"
go run ./cmd/server
```

Then open:

```text
http://localhost:8080/
```

## Notes

- The main application entrypoint is `cmd/server/main.go`.
- The project currently has a single Go source file for the server.
- `go test ./...` reports no test files.
